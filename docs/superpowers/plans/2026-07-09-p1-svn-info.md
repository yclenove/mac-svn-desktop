# P1 Svn Info 实现计划

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务实现此计划。步骤使用复选框（`- [ ]`）语法来跟踪进度。

**目标：** 增加 `svn info --xml` 的基础解析和后端方法，让 P1 工作副本列表可以从真实 WC 读取仓库 URL 与当前 revision。

**架构：** 新增 `SvnInfo` 模型与 `InfoXMLParser`，`SvnCommandBuilder` 负责构造 `info --xml --non-interactive <target>`，`SvnCliBackend` 复用现有 `run` 与错误映射。P3 冲突详情暂不在本切片实现。

**技术栈：** Swift 6.1、Foundation `XMLParser`、XCTest、Subversion CLI 1.14。
